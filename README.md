# 🌟 לוח הכפל - משחק מטבעות

## מה יש בתיקייה זו?
```
index.html   ← המשחק עצמו
manifest.json ← הגדרות PWA (שם, צבע, אייקון)
sw.js        ← Service Worker (עובד גם ללא אינטרנט)
icon-192.png ← אייקון לאנדרואיד
icon-512.png ← אייקון גדול (Play Store)
```

---

## 📋 צעד אחר צעד — העלאה ל-GitHub Pages

### שלב 1 — צור חשבון GitHub (אם אין לך)
1. כנס ל- https://github.com
2. לחץ **Sign up** וצור חשבון

---

### שלב 2 — צור Repository חדש
1. לחץ על **+** בפינה הימנית העליונה ← **New repository**
2. שם: `luch-kefel` (חשוב: באנגלית, בלי רווחים)
3. סמן ✅ **Public**
4. סמן ✅ **Add a README file**
5. לחץ **Create repository**

---

### שלב 3 — העלה את הקבצים
1. בתוך ה-Repository החדש, לחץ **Add file** ← **Upload files**
2. גרור את כל 5 הקבצים מהתיקייה:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. למטה כתוב בהערה: `העלאה ראשונה`
4. לחץ **Commit changes**

---

### שלב 4 — הפעל GitHub Pages
1. לחץ על **Settings** (בתפריט העליון של ה-Repository)
2. בצד שמאל לחץ **Pages**
3. תחת **Branch** בחר: `main` ← `/ (root)`
4. לחץ **Save**
5. המתן כ-2 דקות

---

### שלב 5 — קבל את הקישור
הקישור יהיה בפורמט:
```
https://שם-המשתמש-שלך.github.io/luch-kefel/
```
למשל: `https://shim747.github.io/luch-kefel/`

GitHub יציג לך את הקישור המדויק בדף Settings → Pages

---

## 📱 התקנה על אנדרואיד כ-PWA

### על כל נייד שרוצים:
1. פתח את הקישור ב- **Chrome**
2. לחץ על **שלוש הנקודות** (תפריט) בפינה הימנית העליונה
3. לחץ **"הוסף למסך הבית"** או **"Install app"**
4. לחץ **הוסף** / **Install**

✅ המשחק מופיע כאייקון על המסך הבית  
✅ נפתח במסך מלא בלי שורת כתובת  
✅ עובד גם ללא אינטרנט (אחרי ביקור ראשון)  
✅ המטבעות נשמרים לכל ילד בנפרד על הנייד שלו
